# jacktime
Tidtagningssystem för hundtävlingar
I install mappen finns skript för att installera den virituela miljön som behövs för att köra programet. 
Det finns även en readme fil som förklarar vad du ska göra.

När allt är installerat så aktiverar man miljön och kör main.py med hjälp av python3. 
Jag kör "python3 main.py" i terminalen.
